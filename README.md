# Novexa: Industrial Intelligence & Strategy Platform

Novexa is a dual-engine platform designed to serve two distinct but interconnected markets: small brands seeking accessible business intelligence and large enterprises looking to solve complex R&D challenges. By integrating AI-powered analytics with a global crowdsourcing network, Novexa provides tools for both instant self-diagnosis and long-term strategic execution.

Designed for small brands and D2C businesses, the BI Platform provides a suite of AI-powered calculators to democratize complex analysis without needing a team of data scientists.

- **Customer & Market Sensing**: Synthesize qualitative feedback with quantitative results using the **Review Analyzer** and the **Sales Dashboard** to connect customer sentiment directly to financial performance.

- **Supply Chain Sentinel**: Monitors global news and geopolitical events in real-time to issue warnings about potential disruptions like chip shortages or port strikes, helping secure pricing and avoid delays.

- **Solver Arena**: Structures complex enterprise R&D needs into high-stakes competitions. Global talent competes to build working prototypes for cash prizes and contracts, driving cost-effective and innovative solutions (e.g., *'Reduce Fabric Cutting Waste by 15%'*).

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Novexa.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```
4. Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
